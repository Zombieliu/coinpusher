1. 游戏概述

Numeron Push 是一款
基于 Sui 链 + Dubhe Engine + Discord Activity 的 Web3 推币机互动游戏。

玩家通过投币 → 推币 → 收集金币等道具，以获取：

绑定币（可用于升级、倍率提升、抽奖）

Token（跨社区排行榜奖励）

道具（背包存储）

彩票（用于盲盒抽取）

NFT（赛季奖励）

支持模式：

单人模式（PVE）

社区模式（PVE-P）（服务器一起推进度）

跨服务器赛事（PVP）

目标：

提供强爽感、强留存、强裂变的小游戏体验

作为 NumeronOS / Dubhe 的“互动增长 PaaS 产品”

给 Web3 项目方提供新的社区增长方式

3. 核心玩法：推币系统
3.1 主循环（Core Loop）
投入绑定币 → 推币 → 掉落金币 → 奖励（绑定币 / 道具 / 彩票） → 升级 → 更高倍率 → 再推币

3.2 推币规则

玩家点击按钮投币（自动或手动）

投币掉落金币，推动前方金币掉落边缘

掉落的金币和道具进入奖励槽

有概率触发：

小奖（掉落大量绑定币）

大奖（掉落巨量绑定币 + 彩票）

超级大奖（全服广播 + 彩票雨）

3.3 Jackpot 机制（强成瘾点）

每次投币提升 Jackpot 进度

满值 → 保底出大奖

在 UI 上实时展示进度

4. 单人模式系统
4.1 7 日签到

每日登陆 → 获得绑定币 + 抽奖券

第 7 天奖励翻倍

第 14/30 天送限定皮肤

4.2 每日/每周任务

每日任务例：

推币 300 次

触发小奖 5 次

使用道具 2 次

每周任务例：

推币累计 5000 次

达成大奖 3 次

奖励为：绑定币、经验、抽奖券

4.3 等级系统

等级通过经验获得。

等级奖励：

解锁倍率（1x → 1.2x → 1.5x → 2x）

解锁新皮肤

提升掉落概率

4.4 成就系统

成就示例：

“推币新手” → 推币 1000 次

“大奖猎人” → 获得 10 次大奖

“超级大奖” → 获得全服大奖

奖励：

成就称号

装饰性皮肤

抽奖券

每个服务器有一个 社区进度条：

所有人共同推动

到达进度后 → 全员瓜分奖池 Token

第 1 名社区额外获得 NFT & 称号

奖池 Token 来源：

项目方（合作方 DC）

Numeron 赞助

广告位

5.3 跨服务器赛事（社区 PVP）

周期：每 7 天或 14 天
玩法：

每个 Discord 社区都参与

看谁推的最多

看哪个社区得大奖最多

赛季奖励：

限定 NFT

限定皮肤

社区徽章（展示在 Discord）

5.4 好友助推

每位玩家有一个分享链接：

好友点击 → 给你 +5 绑定币

好友注册 → 双方都有奖励

裂变非常强。

7. 经济系统
6
7.1 游戏内货币
名字	获取方式	用途
绑定币（不可交易）	日常任务、签到、掉落、小奖	推币、升级、抽奖、道具
Token（可交易）	排行榜、社区瓜分、赛季奖励	市场交易、NFT Mint
彩票（可换盲盒）	大奖、活动、任务奖励	抽取皮肤、道具、NFT
7.2 奖池设计（P2E 控制）

单人模式 → 只产出绑定币

Token 只能通过榜单 & 社区获得 → 控制 Token 产出

项目方可注入 Token 奖池 → 激励参与

不会出现“游戏无限产 Token 的崩盘”。

8. 抽奖系统（彩票 → 盲盒）

抽奖内容包含：

道具

皮肤

稀有皮肤

稀有 NFT（赛季限定）

缓冲奖励（必出机制）

抽奖概率完全可控，并且升级盲盒增加更高级奖励。
